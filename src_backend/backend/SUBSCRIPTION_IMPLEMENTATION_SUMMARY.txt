## 🚀 How to Use

### 1. Get All Tiers (For Pricing Page)
```bash
GET /api/profile/subscription_tiers/
```

### 2. Check Feature Access
```bash
POST /api/profile/check_feature_access/
Body: { "feature": "export_products" }
```

### 3. Show Paywall When Access Denied
```bash
GET /api/profile/paywall_data/?feature=api_access
```

### 4. Frontend - Protect Feature
```typescript
<FeatureGate feature="export_products">
  <ExportButton />
</FeatureGate>
```

---

## 🔄 Upgrade Flow

1. **User clicks pro feature** → `check_feature_access` API call
2. **Returns 403 with paywall data** → Display `PaywallModal`
3. **User selects tier** → Navigate to payment
4. **After payment** → Update `subscription_plan` in profile
5. **Next feature check** → Access granted ✅
